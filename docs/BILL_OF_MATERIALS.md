## Quality Considerations

### ESP32 Board Selection
- **Genuine Espressif**: Best compatibility and support
- **DevKit v1**: Most common, well-documented
- **WROOM-32**: Reliable, FCC certified

### Display Quality
- **ILI9341 Controller**: Ensure genuine controller chip
- **Touch Screen**: Optional, not required for this project
- **Viewing Angle**: IPS panels offer better viewing angles

### OBD2 Adapter
- **ELM327 Version**: v1.5 minimum, v2.1 preferred
- **Bluetooth Version**: 2.0 or higher
- **Compatibility**: Check vehicle compatibility list

## Assembly Notes

### Soldering Requirements
- **Breadboard Version**: No soldering required
- **Permanent Version**: Basic through-hole soldering
- **PCB Version**: SMD soldering skills helpful

### Tools Required
- Soldering iron (if permanent assembly)
- Wire strippers
- Multimeter (for troubleshooting)
- Computer for programming
- 3D printer (for custom enclosure)

### Safety Considerations
- Use appropriate fuses for automotive installation
- Ensure proper grounding
- Protect against reverse polarity
- Consider automotive temperature ranges
